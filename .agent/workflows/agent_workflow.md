# 3-step Agent Workflow

When invoked, strictly follow this exact 3-step process for the user's task.

### Step 1: Prompt Enhancer Agent (runs first, automatically)
- Fix all grammar, spelling, clarity issues.
- Make the prompt professional, specific, and AI-optimized.
- Especially strengthen code-related parts (workflow, structure, error handling, best practices).
- Add missing context if needed.
- Show the Enhanced Prompt clearly.

### Step 2: Planning Agent
- Break down the enhanced prompt into clear steps.
- Think step-by-step.
- Ask clarifying questions if anything is still missing.

### Step 3: Execution Agent
- Do the actual work (code, explanation, plan, etc.).
- Follow the plan strictly.
